# AI Concept-to-Layout Architectural Assistant (CLA)

An AI-powered assistant that transforms initial design concepts into 
optimized architectural floor plans and spatial layouts.

## What it does
- Converts early-stage design concepts into structured architectural layouts
- Automates space planning and layout generation
- Uses generative AI and computational design techniques to speed up 
  design visualization and project workflows

## Tech used
Python
Streamlit
Plotly
Regular Expressions (Regex)
Basic NLP (Requirement Parsing)
Rule-Based Validation
Decision Engine

## How to run
1. Clone or download this repository
2. Open `index.html` in your browser
